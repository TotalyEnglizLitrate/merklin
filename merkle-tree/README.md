# Merkle Tree

A simple library for creating Merkle trees. This is a dependency for the Merklin project.

## Installation

This package is intended to be used as a dependency for `merklin` and `arthur` and is not meant to be used independently. It will be installed automatically when you install the client or server.

If you want to install it directly from git, you can use:

```bash
uv add 'merkle-tree @ git+https://github.com/TotalyEnglizLitrate/merklin#subdirectory=merkle-tree'
```
